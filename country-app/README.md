# Country Management System
This is a country management system to display a list of countries.  Countries can be classified as Rainy Country, Snowy Country and Island Country.  At the end, all Snowy Countries are filtered and listed with total snow level in inches.  

## Table of Contents
Features  
Installation  
Usage  
Project Structure  

### Features
Written with HTML and Typescript without frameworks.  For simplicity, no CSS styles are applied in this application.

### Installation
Clone the repository:

bash  
git clone https://github.com/whyanbu/CS601_HW4_YAN.git  
cd CS601_HW4_YAN/country-app/public
Open index.html directly in your web browser.  

No build or dependencies needed.  

### Usage
Start the app by opening index.html.
